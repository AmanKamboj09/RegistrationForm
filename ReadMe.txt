Project Description:
This is a project on Course Registration. A web page is developed using bootstrap, java script(Front Hand) and using Python, Flask(Backend).
The database is designed in mysql. Here the user can enter the particular details and can enroll the students for particular course.

!-------------------------------------------------------------------------------!
Directions to Use:
!-------------------------------------------------------------------------------!

Software Required:- Xampp Server, Python3.6, Notepad++, Pycharm
Python packages Reqiured:- Flask, Flask-sqlalchemy  

!-------------------------------------------------------------------------------!

What about the database????
There is a folder named Database. In this folder there is sql file named dellsoft.sql.
Import this dellsoft.sql file in mysql. 

Now question arises how?????
For this follow some steps:-
1. Start the Xampp Server and run Apache and MySQL.
2. Then open the browser and type localhost in url and press enter.
3. Xampp dashboard will open and then go to phpmyadmin.
4. After that create a new database named dellsoft.
5. After double click on the database and go to imort and select dellsoft.sql

Simple :)  

!-------------------------------------------------------------------------------!

Steps to run the project:-
1. Open Main.py file in pycharm.
2. Open the Xampp Server and start the Apache and MySQL.
3. Now run the Main.py file in pycharm.
4. Click on the IP address  http://127.0.0.1:5000/ 
5. Now a form will be loaded in the default browser.

Hence the project starts........... :)
